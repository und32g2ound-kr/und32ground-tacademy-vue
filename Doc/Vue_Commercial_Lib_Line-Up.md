# Vue 구성요소
 - Instance
    - 뷰로 화면을 개발할 때 필수로 생성해야 하는 단위

 - Component
    - 화면을 구조적으로 설계하기 위한 요소

 - Router
    - 여러개의 화면 간에 이동하는 방법(기술)

 - template
    - 화면을 구체적으로 꾸미는 방법 & 문법
